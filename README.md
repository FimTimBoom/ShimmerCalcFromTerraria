# ShimmerCalcFromTerraria
Я решил себе облегчить работу в игре чтобы бурение шахт не становилось частой рутиной, и создал этот калькулятор для высчитывания Координатов мерцания для удобства себя решил его выложить чтобы вам тоже было удобно, он работает по алгоритму того что координаты мерцания высчитываются также как и выбирается имя 1 гида в мире.
